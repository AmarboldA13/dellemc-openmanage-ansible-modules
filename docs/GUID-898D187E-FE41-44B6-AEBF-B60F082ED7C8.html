View firmware inventory

Command: dellemc_get_firmware_inventory

Synopsis

You can view the firmware inventory of a server using this module. This module displays components of a server and the corresponding firmware versions.

Check_mode support: No

Options

Table 1. dellemc_get_firmware_inventory
Parameter/aliasesRequiredDefaultChoicesComments
idrac_ipYesNANAiDRAC IP Address
idrac_usernameYesNANAiDRAC username
idrac_password/idrac_pwdYesNANAiDRAC user password
idrac_portNo443NAiDRAC port
Table 2. Return Values
NameDescriptionReturnedTypeSample
Firmware Inventory
  • Components of a server and their firmware versions.
  • List of dictionaries, 1 dictionary per firmware.
SuccessStringhttps://github.com/dell/Dell-EMC-Ansible-Modules-for-iDRAC/blob/master/samples/dellemc_get_firmware_inventory.md

Examples

-name: Get Installed Firmware Inventory
   dellemc_get_firmware_inventory:
     idrac_ip:   "xx.xx.xx.xx"
     idrac_user: "xxxx"
     idrac_password:  "xxxxxxxx"
