Configuring System Lockdown Mode

Module: dellemc_system_lockdown_mode
Synopsis

System Lockdown Mode provides a mechanism to protect configuration from any unintentional or accidental changes after the system is provisioned to a certain level.

This module is responsible for enabling or disabling the lockdown mode of a system. When System Lockdown Mode is enabled, the systemâ€™s configuration is locked and system cannot be configured or updated until the lockdown mode is disabled.

Check_mode support: No
